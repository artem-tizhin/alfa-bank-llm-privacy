# NER Masking Service

Этот проект --- сервис для маскирования конфиденциальных данных (PII)
перед отправкой в LLM и последующего восстановления (демаскирования).

------------------------------------------------------------------------

##  Контекст проекта

Проект сделан в рамках соревнования:

https://www.kaggle.com/competitions/llm-march2026-alfabank/overview

------------------------------------------------------------------------

##  Модель

Обученную модель NER можно скачать по ссылке:

https://drive.google.com/file/d/1Tn0k93Luov74cOAqVxSzNftzlYfR45o6/view?usp=sharing

После скачивания её нужно распаковать в:

    artifacts/ner_model/

------------------------------------------------------------------------

## Основная идея

1.  Найти конфиденциальных данные (NER)
2.  Заменить их на плейсхолдеры
3.  Отправить текст в LLM
4.  Вернуть оригинальные данные обратно в ответ

------------------------------------------------------------------------

##  Запуск проекта

### 1. Подготовка

    cd ner_service
    conda create -n ner_service python=3.11
    conda activate ner_service
    pip install -r requirements.txt

------------------------------------------------------------------------

### 2. Настройка `.env`

    cp .env.example .env

И указать:

    LLM_MODE=groq
    GROQ_API_KEY=твой_ключ

------------------------------------------------------------------------

### 3. Модель

Положить в:

    artifacts/ner_model/

Если не положить --- будет работать regex fallback.

------------------------------------------------------------------------

### 4. Запуск

    PYTHONPATH=src uvicorn app:app --reload

Открыть:

    http://127.0.0.1:8000/docs

------------------------------------------------------------------------

##  API

### Swagger

    /docs

### curl

    curl -X POST http://127.0.0.1:8000/process   -H "Content-Type: application/json"   -d '{"prompt": "Моя почта ivan@example.com"}'

##  Пайплайн

### Вход

    Моя почта ivan@example.com и телефон +7 999 123-45-67

------------------------------------------------------------------------

### NER

    [(11, 27, "Email"), (39, 55, "Phone")]

------------------------------------------------------------------------

### Mask

    Моя почта [EMAIL_0] и телефон [PHONE_0]

------------------------------------------------------------------------

### LLM

    Свяжитесь по [EMAIL_0] или [PHONE_0]

------------------------------------------------------------------------

### Unmask

    Свяжитесь по ivan@example.com или +7 999 123-45-67

------------------------------------------------------------------------

##  Итог

    text → NER → mask → LLM → unmask
